# Rhymes & Poetry — Website Rebuild
**Site:** rhymesandpoetry.com  
**Platform:** Wix (existing subscription)  
**Project owner:** Jake Thomas Cohn (@kombuchatea1)

---

## Project Overview

A full redesign of rhymesandpoetry.com — a personal creative brand built around rap, rhymes, and poetry by JTC. The rebuild aims to elevate the visual identity, expand content, improve SEO and traffic, and lay the foundation for future monetization. The current site is a basic Wix blog. The rebuild will modernize it into a full creative brand hub.

---

## Phases

### Phase 1 — Foundation & Redesign (Current)
Core rebuild: new visual identity, improved site architecture, content migration from Instagram, and SEO groundwork.

**Goals:**
- Redesign all existing pages with a modern, editorial aesthetic
- Establish a clear brand identity (typography, color palette, logo refresh)
- Migrate all existing blog posts with proper formatting and tagging
- Import and publish all Instagram content (@rhymesandpoetry_jtc) as site content
- Set up SEO basics: meta titles, descriptions, Open Graph tags, sitemap
- Improve navigation and page structure
- Mobile-first responsive design

### Phase 2 — Traffic & Growth
Content engine + audience building.

**Goals:**
- Launch an email newsletter (Wix Ascend or embedded Mailchimp/ConvertKit)
- Add a Featured Artist submission form — invite others to be featured
- Create a Resources or "Craft" section (writing tips, rap technique breakdowns, form guides)
- YouTube/audio embed support for any recorded content
- Social proof: Instagram feed embed, follower count display
- Consistent content calendar: at least 2 posts/month
- Internal linking strategy for SEO
- Set up Google Analytics / Wix Analytics

### Phase 3 — Monetization
Revenue layer once traffic reaches meaningful levels.

**Potential revenue streams to explore:**
- **Digital downloads**: sell original poem collections, lyric sheets, or writing journals as PDFs (Wix Stores)
- **Merchandise**: print-on-demand (Printful integration with Wix) — t-shirts, hoodies, notebooks with original lines
- **Patreon or membership**: exclusive content for subscribers (early poem drops, behind-the-scenes writing process)
- **Workshops / coaching**: online poetry or rap writing workshops via Wix Bookings
- **Sponsored featured artist posts**: once traffic grows, charge artists for featured placement
- **Affiliate links**: books, music gear, writing tools

## Design Direction

**Aesthetic:** Dark, editorial, literary — think ink and paper meets hip-hop. Not corporate. Raw and intentional.

**Palette (proposed):**
- Background: near-black (`#0f0f0f` or `#111111`)
- Primary accent: warm gold or amber (`#c9a84c`) — nods to classic book spines
- Secondary: off-white/cream (`#f0ead6`) for body text
- Highlight: deep red (`#8b1a1a`) for category tags and CTAs

**Typography (proposed):**
- Headings: a serif with character (e.g., Playfair Display, Cormorant Garamond)
- Body: clean sans-serif (e.g., DM Sans, Inter)
- Accent/pull quotes: monospace (e.g., Space Mono)

**Tone:** Personal, authentic, gritty but thoughtful. The About page voice is the north star — that energy should carry across the whole site.
