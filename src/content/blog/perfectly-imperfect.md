Most of us run a long internal list of *if I always did X, I'd be Y.* The list is mostly fiction. The poem is about retiring it.

<div class="poem-block">

If I always practiced what I preach<br>
I'd truly be a master<br>
If I always did what was right<br>
Maybe I'd learn faster

If I was able to always let go<br>
I would be Buddha's rival<br>
If I were always on time<br>
My name would be in the Bible

If I never acquiesced<br>
I would have no friends<br>
If I never forgave<br>
We'd never make amends

There's a balance in life<br>
That we all must respect<br>
I am me, and you are you<br>
Perfectly imperfect

</div>

Perfection isn't actually the goal. *Balance* is. The if-only person doesn't exist; the actual person, with the actual flaws, is the only person who gets to live the life. Be that one.

<div class="signature">

— JTC
